# Дизайн-концепции для портфолио пейзажного фотографа

## Выбранная концепция: Минимализм с акцентом на фотографию

**Дизайн-движение:** Современный минимализм с элементами фотографического минимализма (Scandinavian Design + Photography-First)

**Основные принципы:**
1. **Фотография в центре** - изображения занимают максимум пространства, текст минимален
2. **Чистота и воздух** - обильное белое пространство, четкая типография, минимум декоративных элементов
3. **Контрастность** - глубокие черные текст и элементы на светлом фоне для четкости
4. **Функциональность** - каждый элемент имеет назначение, нет лишних украшений

**Цветовая философия:**
- **Основной фон:** Белый (`#FFFFFF`) - максимальная чистота и фокус на фотографиях
- **Текст:** Глубокий уголь (`#1A1A1A`) - высокий контраст и читаемость
- **Акценты:** Темно-серый (`#333333`) для вторичных элементов
- **Подсветка:** Светло-серый (`#F5F5F5`) для разделения секций
- **Интерактивные элементы:** Черный с прозрачностью для hover-эффектов

**Парадигма макета:**
- **Асимметричный grid** - фотографии разного размера в галерее (масонри-стиль)
- **Полноширинные секции** - каждая секция занимает всю ширину экрана
- **Вертикальный ритм** - большие промежутки между секциями для дыхания
- **Навигация сверху** - минималистичная, прилипающая при скролле

**Сигнатурные элементы:**
1. **Масонри-галерея** - фотографии разных размеров, создающие динамичный ритм
2. **Полноширинные изображения-разделители** - между секциями для визуального перехода
3. **Минималистичные иконки** - для социальных сетей и контактов

**Философия взаимодействия:**
- **Плавные переходы** - все переходы между состояниями мягкие и естественные
- **Hover-эффекты** - легкое затемнение фотографий при наведении
- **Скролл-анимации** - фотографии появляются при скролле (fade-in)
- **Кликабельность** - социальные ссылки и контакты явно интерактивны

**Анимация:**
- **Entrance animations:** Fade-in при загрузке и скролле (200-400ms)
- **Hover states:** Легкий zoom (1.02x) и затемнение (opacity 0.8) на фотографиях
- **Transitions:** Все переходы 300ms cubic-bezier(0.4, 0, 0.2, 1)
- **Scroll-triggered:** Элементы появляются при входе в viewport

**Система типографии:**
- **Заголовки:** `Playfair Display` (serif, элегантный) - для H1, H2
- **Подзаголовки:** `Inter` (sans-serif, современный) - для H3, H4
- **Тело текста:** `Inter` (sans-serif, 16px, line-height 1.6)
- **Иерархия:** H1 48px bold, H2 32px bold, H3 24px medium, body 16px regular
